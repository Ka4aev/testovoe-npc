# REO Citizens Desk

Frontend test assignment for a citizens service desk dashboard.

## Stack

1. React
2. TypeScript
3. Tailwind CSS
4. Zustand
5. React Router
6. Vite

## Scripts

```bash
npm run dev
npm run build
npm run preview
```

## Current Status

The first implementation stage is complete: the project is initialized with Vite, React, TypeScript, Tailwind CSS, Zustand and React Router.
